# OpenTrace

JART - Just another raytracer

![alt text](Renderer/screenshots/threeColorsMining.ppm)

## Installing

```
git clone --recursive https://github.com/sandrohuber/OpenTrace.git
cd OpenTrace
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=[Debug or Release]
make
```

## Running

```
./Renderer/renderer
```

## Testing

```
./Tests/renderer_test
```
