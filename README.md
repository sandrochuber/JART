# JART

JART - Just another raytracer

![alt text](Renderer/screenshots/threeColorsMining.png)

## Installing

```
git clone --recursive https://github.com/sandrohuber/JART.git
cd JART
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=[Debug or Release]
make
```

## Running

```
./Renderer/renderer
```

## Testing

```
./Tests/renderer_test
```
